## Project Overview
This project focuses on analyzing customer shopping behavior in the retail and e-commerce industry. The primary objective of this analysis is to understand customer purchasing patterns and generate actionable business insights that help improve revenue, customer retention, and marketing performance.

## Business Objective
How can a retail company understand customer purchasing behavior to:Increase overall revenue
- Improve customer retention
- Optimize marketing strategies
- Identify high-value customers
- Improve product performance
- Enhance customer experience

## Dataset Information
Numerical Features
- Age
- Purchase Amount
- Previous Purchases
- Review Rating

Categorical Features
- Category
- Product Name
- Gender
- Season
- Payment Method
- Shipping Type
- Subscription Status
- Discount Applied

## Tools & Technologies Used
- Pandas for Data Cleaning & Transformation
- SQL Server for Data Analysis & Business Query
- Power BI for Dashboard & Visualization

## Project Workflow
1. Data Collection
- Collected retail/e-commerce customer transaction dataset from Kaggle.

2. Data Cleaning & Transformation using Pandas
- The dataset was cleaned and transformed using Python Pandas in Jupyter Notebook.
  
#### Tasks Performed:
- Removed duplicate records
- Handled missing values
- Standardized categorical columns
- Fixed inconsistent data
- Converted data types

#### Exported cleaned dataset to SQL Server for analysis

3. SQL Server Analysis
Business questions were solved using SQL queries in SQL Server.

4. Power BI Dashboard Development
An interactive dashboard was created in Power BI to visualize customer behavior and business performance.

### Key Insights
- Electronics category generated the highest revenue.
- Loyal customers formed the largest customer segment.
- Subscription customers contributed significantly to overall revenue.
- Express shipping customers showed strong revenue contribution.
- Discount strategies strongly influenced purchasing behavior for certain products.
- Older age groups generated higher spending compared to younger customers.
- Customer segmentation helped identify high-value customers.
